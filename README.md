# Java JDBC DAO - Seller & Department

Projeto em Java para praticar o padrão DAO utilizando JDBC e MySQL.
Realiza operações de CRUD para as entidades Seller e Department.

## Tecnologias
- Java
- JDBC
- MySQL
- Padrão DAO

## Estrutura
- Entities: Seller, Department
- DAO interfaces e implementações JDBC
- Conexão via arquivo properties

## Segurança
O arquivo db.properties não é versionado.
Utilize o db.properties.example para configurar o ambiente local.

## Execução
Executar a classe Program após configurar o banco.
